SECTION 1

단일 표본 t-검정 (One Sample t-test)

  • 귀무가설(H0): 모평균 = 특정 값
  • 대립가설(H1): 모평균 ≠ 특정 값 (양측검정)

SECTION 5

ANOVA — 일원분산분석 (집단 비교)

세 개 이상 집단의 평균 차이를 분산을 이용해 검정합니다. (이름은 분산분석이지만 목적은 평균 비교 — 함정 단골!)

  • F = 그룹 간 분산 ÷ 그룹 내 분산 (F-분포 이용)
  • 변수: 독립변수 = 범주형, 종속변수 = 연속형
  • 3대 가정: 정규성 · 등분산성 · 독립성
  • H0: 모든 집단의 평균이 같다 / H1: 적어도 하나는 다르다
예제 5 · 일원분산분석 — 통신사 3사 속도 비교
CONSOLE OUTPUT
  • Df 읽기: 집단 = k−1 (2 → 집단 3개), 잔차 = n−k (42 → n = 45)
  • 같은 분포의 난수라 Pr(>F) ≥ 0.05 → 기각 못함이 정답인 결과 (실행마다 숫자는 다름)
  • 사후검정(Post Hoc): 기각 시 어느 집단이 다른지 찾는 절차 — Tukey(동일 크기 표본), Scheffe·Fisher's LSD(다른 크기 표본)
SECTION 6

ANOVA — 회귀모형의 분산분석표

ANOVA의 두 번째 용도: 회귀 모형에서 설명변수의 유의성 평가. 집단 비교는 aov(), 회귀 분산분석표는 anova(lm객체)를 씁니다.

예제 6 · 회귀 분산분석표 (cars 내장 데이터)
CONSOLE OUTPUT
  • Pr(>F) = p-value → 1.49e-12 < 0.05 → speed는 유의한 설명변수 (*** = p < 0.001)
  • Mean Sq 236.5 = MSE = 오차 분산의 불편추정량 (n이 아닌 자유도로 나눠 불편성 확보)
  • Residuals Df = n − 2 → 48 + 2 = n = 50 ← 계산문제 단골
  • 구조는 집단 비교와 동일: 설명된 변동 ÷ 오차 (그룹 간÷그룹 내 ↔ MSR÷MSE)
